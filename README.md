# Tales of SG

RPG de terminal feito em Python como projeto avaliativo.

O jogador atravessa bairros de São Gonçalo em busca do lendário X-Tudo Supremo.

## Funcionalidades

* Sistema de criação de personagem
* Escolha de classes
* Sistema de batalha
* Sistema avançar entre mapas
* Bosses por mapa
* Inventário do jogador
* Loja em um mapa específico
* Sistema de XP e nível
* Drops de itens
* Tratamento de erros com try/except
* Sistema anti-trapaça com o Cobrador de Impostos Statais

## Como executar

```bash
python tales_of_sg.py
```

É um jogo de terminal feito em Python, então sim, obviamente precisa ter Python instalado.
